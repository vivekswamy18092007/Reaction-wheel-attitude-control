# Reaction-wheel-attitude-control
High-fidelity CubeSat ADCS simulation framework implementing quaternion attitude propagation, rigid body dynamics, reaction wheel control, disturbance rejection, numerical integration, and advanced feedback control for spacecraft stabilization.
